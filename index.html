Welcome


The program files to this app can be downloaded in a zip file above. In order to run this program on your own device (Mac and PC compatible), you must have the latest version of the computation programming language R downloaded. You can find the latest version here: Download and Install R - University of Tennessee Mirror.

Likewise, you will need the latest version of RStudio which you can find here: Download RStudio Desktop.

Running the Langmuir Sorption Modeler on your Mac or PC

Once your aforementioned downloads are complete and installed, to run the app on your computer, simply open RStudio. Then select File -> File Open and navigate to the Zip file you downloaded earlier. After extracting those files, select the file "Langmuir_Shiny_App.R". RStudio will open the program file which you can then run locally on your computer by clicking the green "Run App" button which should launch the interactive Langmuir Sorption Modeler app.

To try out an example of the app functionality, click the Example Sorption Data box. This will load an example dataset of Phosphorus sorption onto soil particles of an Ultisol from collected from Tennessee.

You can upload your own sorption data by selecting the Choose File input. Maximum file size 5MB.
IMPORTANT: Be sure to signify if your data has headers. The default is Headers = FALSE.



After uploading your sorption data, or electing to use the example data, a plot of the molecule or ion sorbed to the sorbent (solid surface) versus the equilibrium concentration of the molecule or ion remaining in solution will be generated. Also displayed below the plot will be outputs of the sorption parameters: Maximum Sorption, the Binding Coefficient, and the Goodness of Fit statistic.

Feel free to change the plot title, and axis labels as necessary to reflect your data. You can even adjust the y-axis limits with the slider option below the plot if the default limits are not adequate. Also you may download a PNG of your plot by clicking the Download button.





You may then select between 5 additional tab panels that provide you with various information.

Tab Panels:

  • Residuals of Fit

      This tab presents the user with a plot of the residuals of the fit of their sorption data to the Langmuir equation i.e. how well does the curve fit the observed data. This is useful for illustrating which data points are ill-conforming.

      NOTE: Do your residuals resemble a "fan-shape"?



      If so, you may want to consider a log-transformation of your dependent variables. This can be done by returning to the "plot-tab" and selecting the "log-transform" checkbox underneath your original plot.

      To learn more about this technique please visit the National Institute of Standards and Technology Engineering and Statistics Handbook.

  • Data

      A reactive table of the user uploaded data. If log-transform Y-axis is selected on the plot-tab, the data displayed here will reflect that change and vice versa.

  • Data Summary

      A statistical summary of the dataset including: the Minimum, 1st Quantile, Median, Mean, 3rd Quantile, and Maximum for the "X" and "Y" column of the data table.

  • File Information

      Basic file path information of the user uploaded dataset.

  • App Credits

      App acknowledgements.
